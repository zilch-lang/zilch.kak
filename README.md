Adds syntax highlighting for both N⋆ and Zilch source code in kakoune.

Syntax highlighting for Zilch is in the [`rc/zilch.kak`](./rc/zilch.kak) file while syntax highlighting for N⋆ is in [`rc/nstar.kak`](./rc/nstar.kak).

## Installing

It is recommended to use [plug.kak](https://github.com/andreyorst/plug.kak) to manage plugins for Kakoune.
You will have to add
```sh
plug "zilch-lang/zilch.kak"
```
